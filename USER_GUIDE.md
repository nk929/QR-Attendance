# 📱 QR 출석 관리 시스템 - 사용 가이드

## 🎯 시작하기

### 1️⃣ 사용자 등록
1. **"사용자 등록"** 메뉴 클릭
2. 등록 항목 커스터마이징 (선택사항)
   - **새 항목 추가**: 학번, 생년월일 등
   - **기존 항목 수정**: 이름, 타입, 아이콘 변경
   - **사용/미사용 토글**: 필요한 항목만 활성화
3. 등록 정보 입력 후 **"등록하기"** 클릭
4. 개인 QR 코드 자동 생성
5. **"다운로드"** 또는 **"인쇄"** 클릭

---

## 📲 모바일에서 QR 스캔하기

### ⚠️ 중요: 데이터 동기화 필수!

LocalStorage는 브라우저별로 독립적이므로, **PC에서 등록한 데이터를 모바일로 전송**해야 합니다.

#### 🔄 데이터 동기화 3단계

##### 1단계: PC에서 데이터 내보내기
```
① "사용자 등록" 페이지로 이동
② "데이터 동기화" 카드에서 "데이터 내보내기" 클릭
③ JSON 파일 다운로드 (예: qr-attendance-backup-2026-02-10.json)
```

##### 2단계: 모바일로 파일 전송
```
방법 A: 카카오톡, 이메일로 파일 전송
방법 B: 클라우드 (Google Drive, Dropbox 등)
방법 C: USB 케이블 직접 연결
```

##### 3단계: 모바일에서 데이터 가져오기
```
① 모바일 브라우저에서 사이트 접속
② "QR 스캔" 페이지로 이동
③ 노란색 안내 박스의 "데이터 가져오기" 버튼 클릭
④ 전송받은 JSON 파일 선택
⑤ "데이터 가져오기 완료!" 메시지 확인
⑥ 이제 QR 스캔 가능! ✅
```

---

### 2️⃣ QR 코드 스캔 (출석 체크)
1. **"QR 스캔"** 메뉴 클릭
2. 모바일인 경우 위 **데이터 가져오기** 먼저 진행
3. 행사/수업명 입력 (선택)
4. **"스캐너 시작"** 클릭
5. 카메라 권한 허용
6. QR 코드를 카메라에 비추기
7. 자동으로 출석 완료!

**✅ 중복 방지 기능**
- 같은 날, 같은 사용자, 같은 이벤트는 한 번만 출석 가능
- 3초 내 같은 QR 재스캔 방지

---

### 3️⃣ 출석 기록 확인
1. **"출석 기록"** 메뉴 클릭
2. 필터링 (날짜, 이름, 행사명)
3. **엑셀 다운로드** (XLSX) 또는 **CSV 다운로드**
4. 개별 기록 삭제 가능

---

### 4️⃣ 대시보드
- 실시간 통계 (등록 사용자, 오늘 출석, 출석률)
- 최근 출석 10건
- 30초마다 자동 새로고침

---

## 🔧 고급 기능

### 📝 입력 항목 커스터마이징
- **추가**: 학번, 생년월일, 주소, 소속 등
- **수정**: 필드 이름, 타입, 아이콘, 필수 여부
- **삭제**: 불필요한 항목 제거 (이름 제외)
- **토글**: 사용/미사용 전환

### 🖨️ QR 코드 인쇄 옵션
- 인쇄할 정보 선택 (이름, 이메일, 전화번호 등)
- QR 크기 선택 (200px / 300px / 400px)
- 인쇄 날짜 포함 여부
- 시스템 로고/이름 포함 여부

### 📊 엑셀 리포트 (XLSX)
- **Sheet 1**: 상세 출석 기록
- **Sheet 2**: 날짜별 통계
- **Sheet 3**: 행사별 통계

## 🐛 문제 해결

### ❌ "등록되지 않은 QR 코드입니다"
**원인**: LocalStorage 데이터 불일치
**해결**:
1. PC에서 "데이터 내보내기" 실행
2. JSON 파일을 모바일로 전송
3. 모바일에서 "데이터 가져오기" 실행
4. QR 재스캔

### ❌ 카메라가 작동하지 않음
**해결**:
1. 브라우저 카메라 권한 허용
2. HTTPS 프로토콜 확인 (필수)
3. 다른 브라우저로 시도

### ❌ 데이터가 사라짐
**원인**: 브라우저 캐시 삭제, 시크릿 모드 사용
**해결**:
1. 정기적으로 "데이터 내보내기"로 백업
2. 일반 모드 사용 권장
